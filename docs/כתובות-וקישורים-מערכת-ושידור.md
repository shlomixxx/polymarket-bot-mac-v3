## מסכי שידור (Stream / OBS)

כל השורות הבאות טוענות את `LiveStreamTrade` במקום `App`.  
פרמטרים:

- **`stream=1`** — Compact (classic)  
- **`stream=2`** — Showcase  
- **`stream=3`** — Spectator (Run P&L, היסטוריית סיבובים, `#stream-stats`)

**מצב שידור מלא־מסך (בלי פסים ריקים בצדדים, מתאים ל־OBS):** הוסף **`&fit=1`** (או **`&broadcast=1`**).

| תיאור | מקומי | Railway (הדוגמה) |
|--------|--------|------------------|
| Compact | `http://127.0.0.1:5175/?stream=1` | `https://polymarket-bot-mac-v3-production.up.railway.app/?stream=1` |
| Showcase | `http://127.0.0.1:5175/?stream=2` | `https://polymarket-bot-mac-v3-production.up.railway.app/?stream=2` |
| Spectator | `http://127.0.0.1:5175/?stream=3` | `https://polymarket-bot-mac-v3-production.up.railway.app/?stream=3` |
| Spectator + קפיצה ל־Run P&L / stats | `http://127.0.0.1:5175/?stream=3#stream-stats` | `https://polymarket-bot-mac-v3-production.up.railway.app/?stream=3#stream-stats` |
| Spectator לשידור (fit) | `http://127.0.0.1:5175/?stream=3&fit=1#stream-stats` | `https://polymarket-bot-mac-v3-production.up.railway.app/?stream=3&fit=1#stream-stats` |

**נתיבים חלופיים (אותה תוצאה לפי `main.tsx`):**

- `/stream/showcase`
- `/stream/spectator`

דוגמה מקומית: `http://127.0.0.1:5175/stream/spectator`

## שידור חי — מה צריך בפועל

1. **מנוע רץ** — מקומית (`npm run dev` / `npm run engine`) או ב־Railway.  
2. **דפדפן או OBS** — טוענים URL של מסך השידור (`?stream=…`, מומלץ `&fit=1` ל־OBS).  
3. **מסך מלא בדפדפן** — אחרי פתיחת הקישור: מסך מלא במערכת (Chrome: View → Enter Full Screen) כדי להסתיר סרגלי דפדפן.

### OBS (Browser Source)

- **URL:** אחד מקישורי ה־Spectator עם `fit=1`, למשל:  
  `http://127.0.0.1:5175/?stream=3&fit=1#stream-stats`  
  (או אותו דבר על דומיין Railway אם השידור לא מהמחשב המקומי).  
- רזולוציה: התאם לפריסה; במצב `fit` התוכן מותאם לגובה viewport.

## מסחר חי (Polymarket) — תזכורת משתני סביבה

פרטים מלאים: `.env.example` בשורש הפרויקט.

| משתנה | משמעות |
|--------|--------|
| `POLYMARKET_PRIVATE_KEY` | מפתח ל־CLOB (אפשר גם דרך הממשק) |
| `POLYMARKET_SIGNATURE_TYPE` | `0` EOA / `1` Poly-proxy וכו' |
| `POLYMARKET_FUNDER` | נדרש לעיתים לחשבון proxy |
| `POLYMARKET_LIVE` | `0` = kill switch (לא שולח פקודות), `1` = מותר אם הממשק מפעיל לייב |

ב־Railway מגדירים משתנים בלוח הבקרה של השירות — לא מעתיקים מפתחות לתוך הקוד.
